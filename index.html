How to Use This Playbook

This is a learning resource, not a test. There are no grades, no scores to report — just practical guidance you can put to work right away.

If you are new to leadership, follow The 12-Week Plan — it sequences the material so you apply one idea a week rather than drowning in ten. If you are here to go deep on a specific topic, use the grid below to jump straight in.

Every section includes frameworks, tips, and practical actions you can apply this week. Small, consistent steps compound into real growth.

Come back to the self-assessment monthly to track where you have been and where you are headed.

Where this sits. This is the third guide in a progression: Curiosity & Commitment builds the apprentice habits, Craft & Ownership grows the senior IC who owns features end-to-end, and this guide picks up when the job shifts from writing code to building the team. If the earlier material still feels like the work you’re doing day-to-day, spend time there first — this one assumes it.
